 Heart Disease Prediction — ML Classification

## Overview
Built 3 ML models to predict heart disease from clinical patient data.
Part of Pluto Academy AI & ML Internship 2026.

## Models Compared
- Logistic Regression — 🏆 Best (F1: 82.35%)
- KNN — F1: 81.16%
- Random Forest — F1: 77.61%

## Key Finding
Logistic Regression outperformed Random Forest on this small, clean dataset —
proving that simpler models often win when data is limited and relationships are linear.
